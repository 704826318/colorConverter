# ColorConverter
颜色格式转换工具

  ## Example

  >rgb和16进制位颜色转换
  ```javascript

  colorConvert('#ffffff') // rgb(255,255,255)

  colorConvert('rgb(255, 255, 255)') // #ffffff

  ```

  > rgba和16进制位转换

  ```javascript

  colorConvert('rgba(255, 255, 255, 1)') // #ffffffff

  colorConvert('#ffffffff') // rgba(255,255,255, 1)

  ```
